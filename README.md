# Azure-Network-Segmentation-Lab

## 📌 Overview
This project demonstrates how to implement secure network segmentation in Azure using VNets, subnets, NSGs, Azure Firewall, and Bastion.

## 🎯 Objectives
- Isolate web, app, and database tiers
- Prevent lateral movement
- Enforce least privilege access
- Implement Zero Trust principles

## 🧰 Technologies Used
- Azure Virtual Network (VNet)
- Network Security Groups (NSGs)
- Azure Firewall
- Azure Bastion
- Azure Monitor
